2. once the account is known to be in the DB (and up to date)
	1. add the UUID of every account in the import list to the WE users follower list (make sure to not add in duplicate UUIDs)
	2. then (if "followed_by_viewer" = true then )
		1. add the UUID of every account in the import list to the WE users follower list (make sure to not add in duplicate UUIDs)
